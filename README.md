## ComboClickMod（连击辅助）

战斗中按右键自动打出可连击的卡牌，按 `` ` `` 键切换模式。

| 模式 | 图标 | 行为 |
|------|------|------|
| 连击 | 飞刀 | 只打高亮可连击卡（普通优先，万能其次） |
| 全自动 | 千刃 | 优先可连击卡，都没有则打最低费卡 |

- 自动跳过消耗卡（检测 `DestroyEffect`）
- 左下水印显示当前模式

**文件：** `mods/ComboClickMod/` | `BepInEx/plugins/ComboClickMod.dll`
